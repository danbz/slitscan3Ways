#Slitscan 3 ways

code in C++ using openFrameworks.cc for three ways slitscannning

included is a compiled binary for OSX

#Controls

* 1: scan horizontal
* 2: scan vertical
* 3: ribbon horizontal
* 4: ribbon vertical
* 5: slitscan clock 
* c: hide preview
* f: toggle fullscreen
* r: draw anystyle scan as radial
